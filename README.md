عنوان: ارائه یک چارچوب بهینه برای طبقه‌بندی تومورهای مغزی با استفاده از یادگیری عمیق و شبکه‌های عصبی کانولوشنی.

مسئله پژوهش:
تشخیص تومورهای مغزی از روی تصاویر MRI به صورت دستی زمان‌بر است و به تجربه پزشک وابسته است. هدف این پایان‌نامه، طراحی یک سیستم هوشمند است که بتواند نوع تومور را به‌صورت خودکار و با دقت بالا تشخیص دهد.

روش انجام کار:

استفاده از تصاویر MRI مغز.
طبقه‌بندی تصاویر در ۴ کلاس:
گلیوما (Glioma)
مننژیوما (Meningioma)
هیپوفیز (Pituitary)
بافت سالم (Normal)
استفاده از یادگیری انتقالی (Transfer Learning) و شبکه‌های عصبی عمیق از پیش‌آموزش‌دیده مانند:
DenseNet121
ResNet50
InceptionV3
Xception
MobileNetV2
ترکیب خروجی این مدل‌ها با روش Stacked Ensemble و یک Meta-Learner برای افزایش دقت تشخیص.

داده‌های مورد استفاده:
مجموعه‌ای شامل ۷۱۵۳ تصویر MRI برای آموزش و ارزیابی مدل استفاده شده است.

نوآوری پژوهش:
به‌جای تکیه بر یک مدل، چند مدل قدرتمند یادگیری عمیق با هم ترکیب شده‌اند تا نقاط ضعف یکدیگر را پوشش دهند و عملکرد کلی سیستم بهبود یابد. همچنین از افزایش داده (Data Augmentation) و یادگیری انتقالی برای کاهش بیش‌برازش و بهبود دقت استفاده شده است.

نتایج:
مدل پیشنهادی به نتایج زیر رسیده است:

Accuracy: 99.30%
Precision: 99.31%
AUC: 0.9998
که نسبت به بسیاری از روش‌های قبلی عملکرد بهتری داشته و می‌تواند به‌عنوان یک ابزار کمک‌تشخیصی برای پزشکان استفاده شود.

هدف نهایی:
کمک به پزشکان و رادیولوژیست‌ها برای تشخیص سریع‌تر، دقیق‌تر و خودکار انواع تومور مغزی از روی تصاویر MRI با استفاده از هوش مصنوعی، به‌منظور کاهش خطای انسانی و تسریع تصمیم‌گیری درمانی.
